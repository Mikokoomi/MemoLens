# MemoLens - UI Screens

## 1. Định hướng giao diện

Giao diện MemoLens nên tạo cảm giác ấm áp, riêng tư, sạch sẽ và giàu cảm xúc. Ứng dụng không nên giống Instagram, Facebook hoặc mạng xã hội.

Phong cách đề xuất:

- Màu nền nhẹ, dễ đọc.
- Khoảng trắng hợp lý.
- Ảnh hiển thị như một phần của câu chuyện cá nhân.
- Nút và form rõ ràng, không gây rối.
- Không có giao diện feed công khai, like, comment hoặc follow.

## 2. Login screen

### Main UI elements

- Logo hoặc tên MemoLens.
- Form nhập email.
- Form nhập mật khẩu.
- Nút Login.
- Link chuyển sang Register.
- Thông báo lỗi nếu đăng nhập thất bại.

### User actions

- Nhập email và mật khẩu.
- Đăng nhập vào tài khoản.
- Chuyển sang màn hình đăng ký nếu chưa có tài khoản.

## 3. Register screen

### Main UI elements

- Tên ứng dụng MemoLens.
- Form nhập họ tên.
- Form nhập email.
- Form nhập mật khẩu.
- Form xác nhận mật khẩu.
- Nút Register.
- Link quay lại Login.

### User actions

- Tạo tài khoản mới.
- Quay lại màn hình đăng nhập.

## 4. Home timeline screen

### Main UI elements

- Thanh điều hướng chính.
- Nút tạo kỷ niệm mới.
- Danh sách kỷ niệm theo dòng thời gian.
- Mỗi item hiển thị ảnh đại diện, tiêu đề, ngày, tâm trạng và địa điểm ngắn.
- Khu vực tìm kiếm hoặc bộ lọc nhanh.

### User actions

- Xem danh sách kỷ niệm cá nhân.
- Mở chi tiết một kỷ niệm.
- Chuyển sang màn hình tạo kỷ niệm.
- Lọc hoặc tìm kiếm kỷ niệm.

## 5. Create memory screen

### Main UI elements

- Form nhập tiêu đề.
- Form nhập câu chuyện hoặc ghi chú.
- Chọn tâm trạng.
- Chọn ngày kỷ niệm.
- Nhập địa điểm.
- Upload một hoặc nhiều ảnh.
- Nhập hoặc chọn tag.
- Chọn album nếu có.
- Nút Save.
- Nút Cancel.

### User actions

- Tạo kỷ niệm mới.
- Upload ảnh từ máy.
- Gắn thông tin cảm xúc và câu chuyện cho kỷ niệm.
- Lưu hoặc hủy thao tác.

## 6. Memory detail screen

### Main UI elements

- Tiêu đề kỷ niệm.
- Ngày, tâm trạng và địa điểm.
- Bộ ảnh của kỷ niệm.
- Nội dung câu chuyện hoặc ghi chú.
- Danh sách tag.
- Album liên quan nếu có.
- Nút Edit.
- Nút Delete.
- Nút quay lại timeline.

### User actions

- Đọc lại toàn bộ kỷ niệm.
- Xem ảnh.
- Chỉnh sửa kỷ niệm.
- Xóa kỷ niệm.
- Quay lại danh sách timeline.

## 7. Edit memory screen

### Main UI elements

- Form chỉnh sửa tiêu đề.
- Form chỉnh sửa câu chuyện.
- Chọn lại tâm trạng.
- Chọn lại ngày.
- Chỉnh sửa địa điểm.
- Quản lý ảnh hiện có.
- Upload thêm ảnh mới.
- Chỉnh sửa tag và album.
- Nút Save changes.
- Nút Cancel.

### User actions

- Cập nhật nội dung kỷ niệm.
- Thêm hoặc xóa ảnh.
- Thay đổi tag hoặc album.
- Lưu thay đổi hoặc hủy.

## 8. Album list screen

### Main UI elements

- Danh sách album cá nhân.
- Mỗi album hiển thị tên, mô tả ngắn, ảnh bìa và số lượng kỷ niệm.
- Nút tạo album mới.
- Nút chỉnh sửa hoặc xóa album.

### User actions

- Xem danh sách album.
- Mở chi tiết album.
- Tạo album mới.
- Chỉnh sửa hoặc xóa album.

## 9. Album detail screen

### Main UI elements

- Tên album.
- Mô tả album.
- Ảnh bìa album nếu có.
- Danh sách kỷ niệm trong album.
- Nút thêm kỷ niệm vào album.
- Nút gỡ kỷ niệm khỏi album.
- Nút quay lại danh sách album.

### User actions

- Xem các kỷ niệm trong album.
- Mở chi tiết một kỷ niệm.
- Thêm hoặc gỡ kỷ niệm khỏi album.
- Quay lại danh sách album.

## 10. Search/filter screen

### Main UI elements

- Ô tìm kiếm theo tiêu đề hoặc nội dung câu chuyện.
- Bộ lọc theo ngày.
- Bộ lọc theo tâm trạng.
- Bộ lọc theo tag.
- Danh sách kết quả tìm kiếm.
- Nút xóa bộ lọc.

### User actions

- Tìm kỷ niệm bằng từ khóa.
- Lọc theo ngày, mood hoặc tag.
- Mở chi tiết kỷ niệm từ kết quả.
- Xóa bộ lọc để xem lại toàn bộ timeline.

## 11. Settings/privacy screen

### Main UI elements

- Thông tin tài khoản cơ bản.
- Tùy chọn đổi tên hiển thị.
- Tùy chọn đổi mật khẩu.
- Ý tưởng bật/tắt privacy lock cơ bản.
- Ghi chú về việc dữ liệu và ảnh là riêng tư theo mặc định.
- Khu vực định hướng tương lai cho export/delete data.

### User actions

- Cập nhật thông tin cá nhân.
- Thay đổi mật khẩu.
- Bật hoặc tắt thiết lập riêng tư cơ bản nếu được triển khai.
- Xem định hướng quyền kiểm soát dữ liệu cá nhân.
